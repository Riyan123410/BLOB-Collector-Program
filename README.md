# Space Game
A game where you collect asteroids wiht your friends for money before the time limit and it gets harder.

## Objective
Fly to the pink planet when you met quota without crashing into obstacles. Each time you make it succesfully the game gets harder

## General
 - WASD to move and E to interact
 - Start the game by pressing the red button at the back of the ship
 - You can customize your caracter in the pause menu by pressing escape

## Ship
- Pink pillar is used for steering
- Green is used to accelerate/go up. The ship will slow down and decrease elevation by default
- Radar system to check home planet and comets
- Yellow pillar to collect rescource planets
- Screen for ship/game information

## Obstacles
 1. Asteroid
    - Basic obstacle that can't move
    - Very common
 2. Comet
    - Hurls toward the ship at very high speeds
    - Can be seen on the radar
    - Rare
 3. Black hole
    - Can't move
    - Pulls the ship towards it
    - Common
